  • The code for the VRDR STU2.2 Death Certification Procedure is SNOMEDCT #103693007 "Diagnostic Procedure (procedure)" . This is a poor choice since it is not descriptive, and it is also current in "inactive" state. This may cause validation errors or warnings on VRDR STU2.2 conformant data. These errors or warnings should be ignored since an inactive SNOMEDCT code remains valid. In VRDR STU3, the code has been changed to a much more descriptive and active code – SNOMEDCT #308646001 Death certification (procedure)
